# tanpura
This website provides a simple and easy-to-use online tanpura that can play different scales and pitches. 
You can adjust the tuning of the tanpura according to your preference. 
The website uses high-quality sound samples and accurate algorithms to create a realistic and harmonious tanpura sound. 
Whether you are a beginner or a professional, you can use this website as a reliable and convenient tool for your music practice or performance.
